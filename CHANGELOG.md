# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.3.2] - 2025-01-10

### Changed

- Simplified version output to show only version number
- Removed verbose config file message from all commands

## [0.3.1] - 2025-01-10

### Added

- Company column to contacts list output

## [0.3.0] - 2025-01-10

### Changed

- Renamed project from hsctl to hscli for better clarity and consistency
- Updated all documentation, commands, and configuration references
- Updated package names and module paths

## [0.1.0] - 2024-01-XX

### Added

- Initial release of hscli
- List contacts with pagination support
- List contact properties
- Create new contacts
- Update existing contacts (including lifecycle stage)
- Delete contacts with confirmation
- Query/search contacts by email or property
- JSON and table output formats
- Version command
- Configuration file support
- Environment variable support for API key
